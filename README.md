# Aplikasi Tes Minat Bakat 
aplikasi tes minat bakat bisa digunakan untuk mempredikasi minat dan bakat siswa, anak, maupun kita yang sudah dewasa

## Persiapan
- PHP 5.6
- Codeigniter 3
- MySql
- Xampp atau Laragon
- Google Chrome (Wajib)
